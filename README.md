# FUTURE_FS_03 - Bean Bliss Café Website

## 🌐 Project Overview
Bean Bliss Café Website is a modern, responsive business website built using React. It simulates a real-world café platform where users can explore menu items, book tables, place orders, and interact with the café digitally.

This project is part of the Future Interns Full Stack Web Development Track (FS).

---

## 🏢 Business Type
Local Café / Restaurant Website

---

## 🚀 Features

### 👤 User System
- User Signup & Login functionality
- Session management using LocalStorage
- Secure access for users

### 🍽️ Menu System
- Dynamic menu display with images and prices
- Add to cart functionality
- Quantity increase/decrease
- Order calculation system

### 🛒 Cart & Orders
- Add/remove items from cart
- Real-time total price calculation
- Order history tracking per user
- Persistent data using LocalStorage

### 🍽️ Table Booking System
- Table reservation feature
- Date and time validation
- Prevents double booking
- Booking history for users

### 📩 Contact System
- Contact form for customer messages
- Stores messages locally

### 🎨 UI/UX Features
- Responsive design
- Smooth animations using Framer Motion
- Interactive and modern UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Framer Motion
- LocalStorage API

---

## 🎯 Project Purpose

This project demonstrates how a real café business can digitize its operations by:

- Allowing customers to browse menu online
- Enabling table reservations
- Managing orders digitally
- Improving customer communication
- Enhancing business presence online

---

## 📊 Business Impact

- Reduces manual booking work
- Improves customer experience
- Increases online engagement
- Helps small cafés build digital presence
- Makes ordering process faster and easier
